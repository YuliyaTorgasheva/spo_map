Обзор дашборда

Этот дашборд показывает информацию о муниципальных районах России. На карте выделены полигоны муниципалитетов, которые можно кликнуть для получения подробной информации.

СПО и UGS

Для каждого муниципалитета отображаются показатели СПО (среднее профессиональное образование) и UGS (учебные группы специальных направлений). Гистограмма визуализирует количество каждой категории.

Использование

Чтобы получить данные по конкретному муниципалитету, кликните на его полигон на карте. Информация отобразится справа вместе с графиком. Можно быстро сравнивать показатели между разными муниципалитетами.

Данные

Все данные загружаются из локальных файлов JSON и GeoJSON, которые находятся в папке data/. Это обеспечивает быстрое отображение без обращения к серверу.
