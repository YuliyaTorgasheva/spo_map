# Дашборд муниципалитетов с анализом доступности СПО

## Описание проекта
Веб-приложение для визуализации муниципалитетов с картой и аналитической панелью. Отображаются полигоны с информацией по доступности среднего профессионального образования (СПО) и наличии укрупненных групп специальностей (УГС).

---

## Наборы данных

### 1. Геоданные муниципалитетов
- Файл municipalities.geojson
- Полигоны муниципалитетов с географическими границами.
- Атрибуты: название, код территории СберИндекс, ОКТМО, население в возрастной группе 15–25 лет и прочие характеристики.
- Используется для отрисовки карты и для отображения агрегированных данных о доступности СПО.

### 2. Колледжи
- Файл colleges.json
- Информация о колледжах: название, код территории из набора МО СберИндекс (territory_id), коды OKTMO, адрес, сайт, сведения из монимторинга СПО Минобрнауки РФ.
- Используется для отображения и подсчёта количества учебных заведений по муниципалитетам.

### 3. УГС СПО
- Файл ugs.json
- Список укрупненных групп специальностей, по которым ведется прием (по состоянию на 2024 год) для каждого колледжа и территории
- org_id - связь с датасетом колледжей, territory_id - связь с датасетом территорий.
- Используется для фильтрации и отображения доступных по регионам групп.

### 4. Соседские муниципалитеты
- Файл territory_neighbors.json
- Описывает соседские связи между муниципалитетами в радиусах 0-50 км, 51-100 км и больше.
- Позволяет расширять область доступности СПО с учетом соседних территорий.
- Структура данных: для каждого муниципалитета по ключу territory_id приведен список соседей в двух категориях: до 50 км и от 51 до 100 км.

## Технологии
- JavaScript (MapLibre GL для карт)
- HTML/CSS (Flexbox для вёрстки)
- JSON для структуры данных

---

## Структура проекта
- `index.html` — главная страница с разметкой и подключением скриптов.
- `main.js` — основной скрипт с логикой загрузки данных, построения карты, фильтров и отображения информации.
- `style.css` — таблица стилей для оформления дашборда, карты, карточек и фильтров.
- Папка `data/` — содержит json и geojson файлы с исходными данными.
